## ğŸ”§ Customization

### Ubah Konten Slide

Edit file `index.html`, cari section dengan `data-slide="X"`:

```html
<section class="slide" data-slide="5">
    <div class="glass-card rounded-3xl p-12">
        <h2>Your Title Here</h2>
        <p>Your content here...</p>
    </div>
</section>
```

### Ubah Warna Theme

Edit Tailwind config di `<head>`:

```javascript
tailwind.config = {
    theme: {
        extend: {
            colors: {
                'neon-blue': '#00d4ff',  // Ubah warna di sini
                'purple-glow': '#8b5cf6',
                // ... dll
            }
        }
    }
}
```

### Ubah Animasi

Edit custom animations di `<style>` tag:

```css
@keyframes glow-pulse {
    0%, 100% { text-shadow: 0 0 20px #00d4ff; }
    50% { text-shadow: 0 0 40px #00d4ff; }
}
```

### Tambah Slide Baru

1. Copy structure slide yang ada
2. Update `data-slide="X"` dengan nomor baru
3. Update `CONFIG.totalSlides` di `script.js`
4. Update total di HTML: `<span id="totalPages">[NEW_TOTAL]</span>`

## ğŸ› Troubleshooting

### Mouse Glow Tidak Muncul
- Mouse glow disabled di mobile (< 768px)
- Check browser console untuk errors

### Keyboard Navigation Tidak Work
- Pastikan focus ada di window (click dulu di slide)
- Check browser console

### Slide Transition Patah-patah
- Cek performa browser
- Tutup tab lain yang heavy
- Coba di browser berbeda (Chrome/Edge recommended)

### Images Tidak Load
- Check koneksi internet (Unsplash images)
- Buka developer tools â†’ Network tab
- Pastikan tidak ada adblocker blocking images
